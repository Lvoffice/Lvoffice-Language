# Lvoffice-Language

Pre-compiled LVLANG binaries and runtime files.

Source code: https://github.com/Lvoffice/lvlangsrc

## Contents

- `bin/lvl.exe` - LVLANG interpreter
- `bin/lvlp.exe` - LVLANG parser
- `libs/lvl_parser.dll` - Parser DLL (Rust)
- `libs/lvl_interpreter.dll` - Interpreter DLL (Rust)
- `grammar.json` - Grammar definition
- `test.lvls` / `test_en.lvls` - Test scripts
